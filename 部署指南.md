
1.下载docker并启动



# 1. 进入项目目录
cd到t3/docker目录

# 2. 启动容器（一键启动 MySQL + Redis）
docker-compose up -d

# 3. 验证状态
docker-compose ps


# 验证新用户
docker exec virtuallive-mysql-dev mysql -u virtual_live -p123 virtuallive_dev -e "SHOW TABLES;"

#idea 连接数据库
docker里的mqsql和redis打开
 填写连接信息
   - `Host`: `localhost`
   - `Port`: `3307`
   - `Database`: `virtuallive_dev`
   - `Username`: `virtual_live`
   - `Password`: `123`
